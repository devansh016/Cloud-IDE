# Cloud-IDE
> #### An online IDE system build using MERN Stack.
## Features
- [x] Execute your code
- [x] Add input file
- [ ] Results shows Time (Sec) and Memory (MB)

## Supported Languages
* C++
* Java
* Python

## Env Variables

Create a .env file in then root and add the following

```
compile_url = your jugde0 url
```
